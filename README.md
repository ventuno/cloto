# Cloto
Play a sound every 30 minutes (only between 7AM and 9PM) to help you feel the passing of time.

## Install
```bash
./install.sh
```

## Uninstall
```bash
./uninstall.sh
```

## Mute
```bash
touch ~/.cloto/MUTE
```

## Unmute
```bash
rm ~/.cloto/MUTE
```
